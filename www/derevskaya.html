ГИС «Исторический атлас Деревской пятины
по писцовой книге письма 1495-1496 гг.»

Эта геоинформационная система была в основном создана еще в 2000-2008 гг. на базе пакета ArcView 3.2 в ходе работы по грантам РГНФ № 00-01-00163а и №03-01-00080а (рук. - А.А. Фролов). Исследовательские результаты работы по проектам, в т.ч. и картографические материалы, опубликованы в трехтомнике: Фролов А.А., Пиотух Н.В. Исторический атлас Деревской пятины Новгородской земли (по писцовым книгам письма 1495-1496 гг.). Тт. 1-3. М-СПб.: Альянс-Архео, 2008. Дальнейшая работа с проектом ведется, как и в остальных случаях, на базе QuantumGIS 1.7-2.8. Публикуемая здесь демоверсия содержит основные слои настольной версии с основными атрибутивными полями. Слои, содержащие данные о селениях и пустошах (кроме центров погостов-округов), по умолчанию невидимы. Их рекомендуется включать только при максимально доступном увеличении - дабы не загромождать окно браузера подписями. Чтобы облегчить восприятие объектов и избежать перегруженности карты, некоторые селения (или их отдельные доли) пришлось вынести на отдельный слой. Список атрибутов, всплывающий при нажатии указателем мышки на место нахождения селения или пустоши, содержит ряд полей.
Ниже приводится их расшифровка.
Toponym - название пункта;
numb_NPK - номер объекта по базе данных, соответствующий номеру в таблице I трехтомника (том 1);
quota - доля селения (в случае, если оно разделено между несколькими владельцами и является «вопчим»);
yards - число дворов в селении; seeding_ps - посев крестьян (в коробьях);
seeding_ln - посев землевладельца и/или его людей;
hay_peasan - число копен сена крестьян;
hay_lndwnr - число копен сена землевладельца и/или его людей;
obji_total - суммарный обежный оклад;
obji_psnt - обежный оклад крестьян;
obji_lndwn - обежный оклад землевладельца и/или его людей; 

numb_lndwn - номер землевладения по базе данных, соответствующий номеру в таблице II трехтомника (том 1);
uezd_dacha - уезд и номер земельной дачи Генерального межевания конца XVIII в., в пределах которой локализован объект писцовой книги;
GM_part - номер части соответствующего уездного плана;
old lndwnr - «старый землевладелец», упомянутый писцовой книгой в прошедшем времени;
lndwnr1499 - владелец, которому принадлежала земля на момент завершения писцовой книги (1499 г.);
status - статус локализованного пункта по писцовой книге.

По умолчанию невидим и еще один слой, "contemporary raion division”. На нем отображено современное деление по областям и районам, помогающее в ориентации на исторической карте. Всплывающий при нажатии указателем мышки список атрибутов содержит сведения о названии района, коде ОКАТО, площади его территории, принадлежности к федеральному округу и области.
Обращаем внимание пользователя, что в силу технических ограничений публикуемой здесь демо-версии ГИС по нажатию мышки отображается список атрибутов самого нижнего из всех видимых полигональных слоев карты (то есть расположенных последним в перечне слоев в правом верхнем углу окна карты).
В публикуемой версии ГИС полигональных слоев всего два: 1) "pogost borders” с реконструированными границами погостов-округов XV в. и 2) "contemporary raion division” - современные административные районы). Если в точке карты, где произошло нажатие указателем мыши, находятся объекты обоих слоев и оба слоя включены, то список атрибутов будет доступен только для слоя 2). Чтобы увидеть атрибуты полигонального слоя 1), необходимо сделать невидимым слой 2).

А.А. Фролов
